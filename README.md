# Homebrew Tap

This is a Homebrew tap for my personal projects.

## Installation

```bash
brew tap agustinvalencia/tap
```

## Available Formulae

### mdvault

CLI tool for managing markdown vaults with structured notes, validation, and search.

```bash
brew install agustinvalencia/tap/mdvault
```

Or directly:

```bash
brew install agustinvalencia/tap/mdvault
```
